# Projet Java : Incrémentation, Tri et Recherche

## 📌 Description
Ce projet a été réalisé dans le cadre du cours de programmation Java.  
Il comporte deux parties principales :
1. **Présentation PowerPoint (mini-cours)** : explication des concepts d’incrémentation en Java et des algorithmes classiques de tri et de recherche.
2. **Implémentation Java** : code source des exercices d’incrémentation, des algorithmes de tri (sélection, bulles, insertion) et des algorithmes de recherche (linéaire, binaire).
